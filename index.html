Ethical Hacking

Ethical hacking is an authorized practice of detecting vulnerabilities in an application, system, or organization’s infrastructure and bypassing system security to identify potential data breaches and threats in a network


Ethical hackers aim to investigate the system or network for weak points that malicious hackers can exploit or destroy. They can improve the security footprint to withstand attacks better or divert them.

Techniques

This section explores the techniques used in each phase of ethical hacking.

OSINT (Open Source Intelligence)

Leveraging publicly available sources of information, such as public records, domain registration details, and social media profiles, to build a comprehensive profile of the target.

Reconnaissance

WHOIS Lookups

Querying WHOIS databases to obtain information about domain ownership and registration details.

Scanning

Port Scanning

Port scanning, in the context of cybersecurity, is the practice of probing a network or server to identify open ports. These ports are essentially virtual doorways that allow communication between devices on a network. By scanning for open ports, attackers can gain valuable information about the target system.

Scanning

Vulnerability Scanning

Vulnerability scanning, often shortened to "vuln scanning," is another crucial cybersecurity practice that works hand-in-hand with port scanning. While port scanning identifies open ports and potential services, vulnerability scanning goes a step further. It analyzes these open ports and services to identify known security weaknesses and flaws that attackers could exploit.

Scanning

Banner Grabbing

A technique used to gain informations about the computer system on a network and the services running on its open ports. Administrators can use this to take inventory of the systems and services on their Network. However an intruder can use Banner grabbing in order to find network hosts that are running versions of applications and operating systems with known exploits.

Scanning

Network Mapping

Finding the topology of network, routers, firewalls servers if any, and host information and drawing a network diagram with the available information. This map may serve as a valuable piece of information throughout the haking process.

Gaining Access

Exploiting Software Vulnerabilities

Ethical hackers may attempt to exploit known software vulnerabilities in operating systems, applications, or services running on the target system. This can involve techniques like SQL injection, or remote code execution.

Gaining Access

Brute-forcing

This method involves systematically trying all possible combinations of username and password until a match is found. While time-consuming, it can be effective against weak passwords.

Gaining Access

Dictionary attacks

Similar to brute-forcing, but uses a pre-defined dictionary of common words and phrases instead of trying every combination.

Gaining Access

Social engineering

Attackers can trick users into revealing their passwords through deceptive tactics like phishing emails or phone calls.

Maintaining Access

Backdoors

Backdoors provide a means for attackers to maintain persistent access to a compromised system. Unlike some other methods of unauthorized access, backdoors are designed to remain undetected over time, allowing attackers to come and go without being noticed.

Clearing Tracks

Log Deletion

log deletion refers to the removal of records of system events, activities, or user actions. These logs can be found in various forms, such as text files, database records, or even specific log files within applications.

Clearing Tracks

Anti-Forensic Techniques

An intelligent hacker always clears all evidence so that in the later point of time, no one will find any traces leading to him. This involves modifying/corrupting/deleting the values of Logs, modifying registry values and uninstalling all applications he used and deleting all folders he created.
